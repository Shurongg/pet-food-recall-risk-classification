# Pet Food Recall Risk Classifier — Hugging Face Space

An interactive demo for a multi-label text classifier that assigns risk
categories to pet food recall and safety-alert records.

Built as part of an academic Information Retrieval course project.

---

## What this Space does

Accepts a pet food brand name, product description, and recall reason as free
text, then predicts one or more of the following risk categories:

| Label | Description |
|---|---|
| `PATHOGEN_CONTAMINATION` | Bacterial or pathogen-related contamination (Salmonella, Listeria, E. coli, etc.) |
| `CHEMICAL_OR_NUTRITIONAL_RISK` | Chemical, mycotoxin, heavy metal, or vitamin/mineral imbalance risks |
| `PHYSICAL_OR_QUALITY_ISSUE` | Foreign material, labeling errors, packaging defects, or process-control failures |

The app shows predicted labels with confidence scores, per-label explanations,
and the exact input text fed to the model.

---

## Model

The classifier is a `OneVsRestClassifier(LogisticRegression)` trained on
sentence-transformer embeddings from `sentence-transformers/all-MiniLM-L6-v2`.
The encoder is frozen; only the sklearn head is stored in the model repository.

- Model repository: https://huggingface.co/ShurongSR/pet-food-recall-risk-classifier
- Dataset repository: https://huggingface.co/datasets/ShurongSR/pet-food-recall-risk-classification

---

## Input format

The app constructs the following string before encoding:

```
Brand: {brand}. Product: {product}. Recall reason: {reason}.
```

## Disclaimer

This tool is for educational purposes only. Predictions are generated by a
machine learning model trained on a small academic dataset and may be incorrect.
Always consult official FDA recall notices and a qualified veterinarian for
authoritative pet food safety information.
